XML JSP Tag Library

demo/taglib/xjtldemo.war is a Java Web Application that uses XML JSP Tag Library (XJTL) and JSP Standard Tag Library (JSTL). It was tested with Apache Tomcat 4 + JDK 1.4 and with Apache Tomcat 4 + Xalan + JDK 1.3.

All XML Output and XML Processing examples are fully documented.

Note: Currently, XJTL supports JSP 1.2 and uses Apache JSTL 1.0 to evaluate the ${...} expressions. Therefore, you have to deploy XJTL together with JSTL. The next version of XJTL will be based only on JSP 2.0. The XJTLDEMO web app contains the JAR files of JSTL 1.0 and it is assumed that the JAXP 1.1 implementation is provided by the JDK or app server. If you don't use JDK 1.4, make sure that your classpath contains a JAXP 1.1 parser (Xerces or Crimson) and the Xalan XSLT processor.

SAXDOMIX Framework

demo/saxdomix contains examples based on the SAXDOMIX framework. To run the examples, execute the *.bat or *.sh files from demo/saxdomix. If you don't use JDK 1.4, make sure that your classpath contains a JAXP 1.1 parser (Xerces or Crimson) and the Xalan XSLT processor.

XML Parsing Benchmark

demo/benchmark contains the *.bat files for running the benchmark. You'll have to edit initEnv.bat and set the JDK and parser paths. Visit http://www.devsphere.com/xml/benchmark/index.html for charts, results and conclusions.
